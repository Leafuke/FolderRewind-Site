# Quick Start

Welcome to **FolderRewind**! This guide will help you go from installation to your first backup in just 5 minutes.

## What is FolderRewind?

FolderRewind is a modern folder backup manager built with WinUI 3. It can:

- Use the **7-Zip engine** for efficient compression and encrypted backups
- Support **scheduled / interval / on-startup** automation modes
- **Rewind** to any backup point via the History Timeline
- Extend functionality for specific scenarios (e.g., Minecraft) through the **plugin system**

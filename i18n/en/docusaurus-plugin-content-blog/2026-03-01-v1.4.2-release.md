## FolderRewind v1.4.2

v1.4.2 brings stronger plugin extensibility, better history management, and multiple quality-of-life improvements.

<!-- truncate -->

## Update Content

1. Support for plugins to modify configuration Config.
2. Support for plugins to modify post-restore behavior.
3. Support for displaying more historical node information.
4. Support for marking history records as "Important," which will prevent them from being automatically deleted.
5. Support for automatic update detection.
6. Support for the "Rebuild History" feature. You can select a folder containing backup files and generate history records based on it. This effectively prevents accidental history deletion.
7. Support for the MARK_IMPORTANT remote command.
8. Improved functionality for modifying folder descriptions.
9. Fixed the issue where silent startup could not be set.
10. Other minor bug fixes.

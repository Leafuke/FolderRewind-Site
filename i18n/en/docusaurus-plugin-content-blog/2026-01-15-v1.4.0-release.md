## FolderRewind v1.4.0

v1.4.0 is a major update introducing KnotLink integration support and many feature enhancements.

<!-- truncate -->

### New Features

- **KnotLink IPC Protocol**: A new inter-process communication protocol for deep integration with third-party tools
- **Linkage Module Support**: Plugins can now implement more complex interaction scenarios through KnotLink
- **MineRewind v1.4.0**: Minecraft save plugin updated with linkage module support

### Upgrade Notes

- It is recommended to use the **Data Migration** feature in settings to back up your configurations and history before updating
- To use the linkage module feature, ensure:
  - Linkage module version >= 1.0.0
  - MineRewind plugin version >= 1.4.0
  - KnotLink server is downloaded

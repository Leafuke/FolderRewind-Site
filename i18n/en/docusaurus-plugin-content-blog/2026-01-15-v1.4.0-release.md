## FolderRewind v1.4.0

v1.4.0 is a major update introducing many feature enhancements.

<!-- truncate -->


## Update Content

1. Upgraded to dotnet10 (previously dotnet8).
2. New configuration type - Encrypted configuration. When creating this configuration, you will have a one-time opportunity to set a password. Be sure to remember this password, as it will be required for each restoration.
3. Support for custom handling logic for different files. For example, you can set `.mp4` files to "Store Only" mode to reduce additional time consumption.
4. Support for plugins to invoke connectivity features.
5. Support for more advanced scheduled tasks.
6. Support for setting default backup storage path. Independent configurations will automatically set their backup storage path to `default path/configuration name`.
7. Support for pinning folders.
8. Improved connectivity strategy.
9. Enhanced task interface. Now supports displaying progress and error reasons.
10. Improved startup guidance.
11. Enhanced error notifications.
12. Improved mini floating window functionality.
13. Enhanced restoration search.
14. Fixed adjustable range issues for compression levels corresponding to different compression algorithms.
15. Fixed multiple potential crash issues.
16. Other minor fixes.

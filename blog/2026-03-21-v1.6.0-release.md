## FolderRewind v1.6.0

v1.6.0 聚焦于“可验证、可复用、可分享”的备份体验，引入核心功能自动校验、配置模板工作流与更完整的模板分享链路。

<!-- truncate -->

:::tip 下载建议
推荐优先使用 [Microsoft Store](https://apps.microsoft.com/detail/9nwsdgxdqws4) 安装。

如果你使用的是侧载版本，v1.6.0 已支持设置 GitHub 镜像源，可改善更新与在线模板相关体验。
:::

## 更新内容

1. 支持自动校验核心功能。
2. 支持创建和使用配置模板。
3. 支持模板的分享与导入。
4. 支持设置 GitHub 镜像源。
5. 支持边栏自动折叠。
6. 优化部分界面显示效果。
7. 优化部分页面的加载性能。
8. 优化侧载版本更新体验。
9. 修复潜在的路径越界问题。
10. 修复设置卡片动画问题。
11. 修复安全删除失败的问题。
12. 其他小问题修复。

## 本次更新重点

### 核心功能自动校验

设置页新增了核心功能自动校验能力。它会自动创建临时环境，并验证：

- 初始全量备份
- 无变化跳过
- 智能增量备份
- 共享锁文件备份
- 清洁还原与覆写还原
- 安全删除
- 数量限制与自动清理

对于刚升级、刚换机器或准备正式启用自动备份的用户，这会非常实用。

### 配置模板工作流

现在你可以把成熟配置保存为模板，并且：

- 从模板创建配置
- 自动识别候选来源文件夹
- 编辑和管理本地模板
- 在不同设备间复用同一套备份策略

这让 FolderRewind 更适合多项目、多设备和社区共享场景。

### 模板分享与镜像源

v1.6.0 补齐了模板导出、导入、官方模板浏览、分享码导入以及 GitHub 提交流程。

同时，侧载版本新增 GitHub 镜像源设置，在线模板与更新检查也会受益。

## 建议升级后做的事

- 在设置页运行一次 **核心功能自动校验**
- 为你最常用的一套配置创建模板基线
- 如果使用侧载版本，检查是否需要切换 GitHub 镜像源

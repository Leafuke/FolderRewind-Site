# 快速开始

欢迎使用 **FolderRewind（存档时光机）**！本指南将帮助你在 5 分钟内完成从安装到首次备份的全部流程。

## 什么是 FolderRewind？

FolderRewind 是一款基于 WinUI 3 的现代化文件夹备份管理工具。它能够：

- 使用 **7-Zip 引擎** 高效压缩和加密备份
- 支持 **定时/间隔/启动时** 三种自动化备份模式
- 通过 **历史时间轴** 回溯到任意备份时间点
- 通过 **插件系统** 为特定场景（如 Minecraft）深度优化

## 快速上手三步走

### 第一步：安装

从 Microsoft Store 获取（推荐），或参考 [安装指南](./getting-started/installation) 进行侧载安装。

<a href="https://apps.microsoft.com/detail/9nwsdgxdqws4" target="_blank">
  👉 从 Microsoft Store 安装
</a>

### 第二步：创建配置并添加文件夹

1. 启动 FolderRewind，点击主界面的 **「+」** 按钮。
2. 选择配置类型（通常选择 **Default**），为配置命名。
3. 进入配置详情，点击 **「添加文件夹」**，选择要保护的文件夹路径。

> **Minecraft 玩家？** 安装 [MineRewind 插件](./guides/minecraft)后，可自动扫描并发现所有存档文件夹。

### 第三步：执行首次备份

在配置详情页面点击 **「立即备份」** 按钮。备份完成后，你可以在历史时间轴中看到这次备份记录。

🎉 **恭喜！** 你的文件夹已受到保护。

# 黑名单与还原白名单

过滤规则是“省空间”和“防误恢复”的关键工具，但也最容易因为规则过猛导致数据缺失。

## 备份黑名单

你可以排除不需要进入压缩包的文件/目录，用于：

- 减少体积
- 降低噪音文件影响
- 缩短备份时间

常见排除项示例：日志、缓存、临时目录。

### 推荐先排除的类型

- 运行时日志目录
- 临时缓存目录
- 可再生的中间文件

## 还原白名单

还原时可通过白名单控制仅恢复特定项，适合精细回滚。

适用场景：

- 只恢复建筑区域相关文件
- 只回滚配置文件，不动玩家进度
- 验证某个可疑文件是否导致问题

## 实践建议

- 黑名单先小范围试跑，避免误排除关键数据
- 自定义还原时优先在测试世界验证规则

## 安全策略

1. 新规则先在测试配置验证
2. 连续两次备份结果都正常再上生产配置
3. 遇到异常先回退规则，不要继续叠加复杂条件

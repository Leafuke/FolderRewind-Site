# 数据迁移

FolderRewind 支持在设置页导入导出两类数据：

- 配置数据（`config.json` 结构）
- 历史记录数据（`history.json` 结构）

## 入口

1. 打开 **设置**。
2. 找到 **数据迁移** 区块。
3. 按需选择导出或导入。

![数据迁移区域中的导出与导入按钮](/img/docs/guides/data-migration-import-export.png)

## 配置迁移

### 导出配置

1. 点击 **导出配置**。
2. 选择保存路径（`.json`）。
3. 等待成功提示。

### 导入配置

1. 点击 **导入配置**。
2. 阅读确认提示并继续。
3. 选择配置 JSON 文件。

导入配置会替换当前配置内容；系统会先尝试备份旧配置为 `.bak`。

## 历史记录迁移

### 导出历史

1. 点击 **导出历史**。
2. 选择保存路径（`.json`）。

### 导入历史

1. 点击 **导入历史**。
2. 选择模式：
   - **合并**：仅导入不存在的条目（去重）
   - **替换**：用导入文件覆盖当前历史
3. 选择历史 JSON 文件并执行。

替换模式会先尝试备份现有历史为 `.bak`。

## 建议迁移顺序（新电脑）

1. 先安装并启动 FolderRewind。
2. 导入配置。
3. 导入历史（通常先用合并）。
4. 打开几个配置验证源路径和目标路径是否可访问。
5. 手动执行一次备份与一次恢复测试。

## 注意事项

- 跨设备时，路径盘符或目录结构可能不同，导入后需检查路径有效性。
- 加密配置依赖本机密码存储机制，迁移后请优先验证还原流程。

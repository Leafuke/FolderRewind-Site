# 自动化任务

FolderRewind 支持三种自动化备份模式，让你"设置一次，永久运行"。

## 自动化模式

### 间隔备份

按固定时间间隔自动执行备份。

- **配置方式：** 进入配置详情 → 自动化设置 → 选择「间隔备份」
- **间隔范围：** 最短 1 分钟，推荐 15~60 分钟
- **适用场景：** 游戏存档、频繁变动的工作文件

### 定时备份

设置每日固定时间点执行备份（24 小时制）。

- **配置方式：** 进入配置详情 → 自动化设置 → 选择「定时备份」
- **示例：** 每天 02:00 执行夜间备份
- **适用场景：** 不需要频繁备份的稳定数据

### 启动时备份

在 FolderRewind 启动时自动执行备份。

- **配置方式：** 进入配置详情 → 自动化设置 → 选择「启动时备份」
- **适用场景：** 确保每次使用前数据是最新状态
- **提示：** 将 FolderRewind 设为开机自启，实现完全无感备份

## 组合使用

你可以为同一个配置同时启用多种自动化模式。例如：

- ✅ 启动时备份 + 每 30 分钟间隔备份
- ✅ 启动时备份 + 每天 22:00 定时备份

## 保留策略

自动化备份会产生大量备份文件。建议配置保留策略：

- 保留最近 N 个备份
- 自动清理超过指定天数的旧备份

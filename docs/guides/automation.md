# 自动化任务

FolderRewind 可以为每个配置独立开启自动备份，支持：

- 间隔模式
- 计划模式（按月/日/时/分条目）
- 启动时触发
- 连续无变更自动停

## 开始前确认

- 该配置已添加至少一个文件夹。
- 目标备份路径已设置且可写。
- 建议先手动完成一次备份，确认流程正常。

## 配置入口

1. 在首页进入目标配置。
2. 点击 **配置设置**。
3. 打开 **自动化** 选项卡。
4. 开启 **启用自动备份**。

![自动化选项卡中的启用自动备份开关](/img/docs/guides/automation-enable-auto-backup.png)

## 自动化方式详解

### 间隔备份

按固定分钟间隔执行自动备份。

- **配置项：** `周期运行间隔`（分钟）
- **范围：** `1 ~ 10080` 分钟
- **建议：** 常规用户从 `30` 或 `60` 分钟起步

### 计划备份（Scheduled Mode）

开启 **计划备份模式** 后，可添加多个计划条目。

每个条目可设置：

- 月（可选“每月”）
- 日（可选“每天”）
- 时、分

常见示例：

- 每天 `02:00`
- 每月 `1` 号 `03:30`
- 每月 `1` 号与 `15` 号各一条

### 启动时备份

开启“程序启动时运行”后，每次应用启动会为该配置触发一次自动备份。

- **适用场景：** 你每天首次打开应用时希望先做一次快照
- **建议：** 与间隔模式搭配使用

## 连续无变更自动停止

可开启 **重复未发现变更后停止自动任务**：

1. 打开该开关。
2. 设置阈值（连续 N 次无变更后停止）。

行为说明：

- 连续检测到“无变更”达到阈值后，该配置的自动备份会自动关闭。
- 一旦出现变更，连续计数会重置。

## 组合使用

你可以为同一个配置同时启用多种自动化模式。例如：

- ✅ 启动时备份 + 每 30 分钟间隔备份
- ✅ 启动时备份 + 每天 22:00 计划备份
- ✅ 间隔备份 + 连续无变更自动停止

## 推荐参数（可直接套用）

### 通用文档

- 间隔：`60` 分钟
- 启动时备份：开
- 连续无变更自动停：开，阈值 `5`

### Minecraft 存档

- 间隔：`15~30` 分钟
- 启动时备份：开
- 可额外加一条夜间计划备份（如 `03:00`）

## 配套保留策略（很重要）

自动化备份会产生大量备份文件。建议配置保留策略：

- 在“备份策略”中设置 **保留最近备份数**。
- 使用智能增量时，同时设置 **链长度上限**，控制还原链复杂度。

## 排查要点

- 任务没触发：确认该配置“启用自动备份”是否仍为开启。
- 计划模式无效：确认计划条目的小时/分钟是否设置正确。
- 自动停止后不再执行：这是“无变更自动停”生效，需手动重新开启自动备份。

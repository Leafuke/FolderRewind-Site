# 备份模式详解

FolderRewind 基于 7-Zip 引擎提供多种备份方式，满足不同场景需求。

## 备份模式对比

| 模式 | 说明 | 优势 | 适用场景 |
|------|------|------|---------|
| **完整备份** | 每次备份全部文件 | 独立完整，还原简单 | 文件量小、关键时间点 |
| **增量备份** | 仅备份自上次备份以来变更的文件 | 速度快，占用空间少 | 大型文件夹的日常备份 |
| **差异备份** | 备份自上次 *完整备份* 以来变更的文件 | 还原只需完整包 + 最新差异包 | 平衡速度与还原便利性 |

## 选择建议

- **Minecraft 存档：** 完整备份（存档大小通常可控）
- **工作文档/项目：** 增量备份（文件频繁变动，节省空间）
- **重要资料归档：** 完整备份 + 加密

## 压缩设置

FolderRewind 使用 7z 格式进行压缩，你可以在配置中调整：

- **压缩等级** — 从「存储」到「极限」多个级别
- **加密** — 可选 AES-256 加密保护

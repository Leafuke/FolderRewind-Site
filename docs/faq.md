# 常见问题 (FAQ)

## 安装相关

### FolderRewind 支持哪些操作系统？

Windows 10 1809 (17763) 及以上版本，包括 Windows 11。支持 x64 和 ARM64 架构。

### Microsoft Store 安装和侧载安装有什么区别？

- **Store 安装（推荐）：** 自动更新，安全沙箱运行，一键安装。
- **侧载安装：** 需要手动启用开发者模式，适合无法访问 Store 的用户。

### 安装后无法启动怎么办？

1. 确认系统版本满足最低要求
2. 重新安装最新版本（Store 或 Release）
3. 检查是否被安全软件拦截
4. 前往 [GitHub Issues](https://github.com/Leafuke/FolderRewind/issues) 搜索或提交问题

## 备份相关

### 备份文件存放在哪里？

每个配置都有自己的目标路径。你可在“配置设置”中查看或修改目标路径。

### 备份会占用太多磁盘空间吗？

可通过以下方式控制体积：

- 使用智能增量模式
- 开启无变更跳过
- 设置“保留最近备份数”
- 对大文件类型设置更低压缩策略

### 支持备份正在运行的游戏存档吗？

配合 MineRewind 插件，支持在 Minecraft 运行中进行热备份。对于其他游戏，建议在游戏暂停或关闭后再执行备份。

### 备份过程中可以继续使用电脑吗？

可以。备份在后台执行，不影响正常使用。

### 为什么自动备份突然停止了？

可能启用了“重复未发现变更后停止自动任务”。达到阈值后会自动关闭该配置的自动备份，需要手动重新开启。

## 还原相关

### 还原操作会覆盖当前文件吗？

当前主要有两种还原模式：

- **Clean 还原（清空目标）**：先清理再还原，最推荐
- **Overwrite 还原（覆盖）**：覆盖同名文件，可能保留旧文件

建议在关键目录启用“还原前自动备份”。

### 可以只还原部分文件吗？

当前版本支持整文件夹还原。部分文件还原功能在规划中。

### 为什么点击“恢复”后提示输入密码？

该配置是加密配置（Encrypted）。还原前必须通过密码验证。

### 为什么历史里有记录，但“查看”找不到备份文件？

通常是压缩包已被手动移动或删除。可在历史页面使用“清除失效”清理无效条目。

## 数据迁移相关

### 配置和历史可以迁移到新电脑吗？

可以。设置页支持导出/导入配置与历史记录。

### 导入历史时“合并”和“替换”有什么区别？

- **合并**：仅导入不存在的条目（去重）
- **替换**：用导入文件覆盖当前历史

替换前系统会尝试备份旧数据为 `.bak`。

### 加密配置跨设备迁移要注意什么？

加密密码为本机用户环境下的安全存储数据。迁移后请优先验证加密配置的还原流程。

## 插件相关

### 如何安装插件？

可在设置页插件区手动安装（zip），也可打开插件目录进行管理。

### MineRewind 插件是免费的吗？

是的，MineRewind 是官方发布的免费开源插件。

### 如何开发自己的插件？

请参考 [插件开发快速上手](/docs/plugins/developing/quick-start) 文档。

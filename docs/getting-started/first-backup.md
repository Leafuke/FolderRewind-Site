# 首次备份

本教程将引导你完成第一次备份。

## 前提条件

- 已完成 [安装](./installation)

## 操作步骤

### 1. 创建配置

1. 启动 FolderRewind，进入主界面。
2. 点击右下角的 **「+」** 按钮。
3. 选择配置类型：
   - **Default** — 通用备份，适合大多数场景
   - 如已安装 MineRewind 插件，还会出现 **Minecraft** 类型
4. 为配置命名（如「工作文档」或「MC 存档」）。

### 2. 添加文件夹

1. 进入刚创建的配置详情。
2. 点击 **「添加文件夹」** 按钮。
3. 选择你要保护的文件夹路径。
4. 可重复此步骤添加多个文件夹。

> **提示：** 每个文件夹是软件操作的最小单元。一个配置可包含多个文件夹，它们共享同一套备份规则。

### 3. 执行备份

点击配置详情页面的 **「立即备份」** 按钮。

备份过程中，你可以看到进度条显示压缩进度。完成后，在 **历史时间轴** 中会出现这次备份记录。

### 4. 验证备份

在历史时间轴中点击刚创建的备份条目，可以预览备份内容，确认文件完整。

## 进阶技巧

- **收藏文件夹：** 将高频使用的文件夹标记为收藏，方便在主界面快速操作。
- **快捷键备份：** 配置全局快捷键，无需打开主窗口即可执行备份。
- **Mini 悬浮窗：** 为特定文件夹创建迷你窗口，边工作边监控。

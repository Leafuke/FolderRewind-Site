# 首次备份

本教程将引导你从零创建配置、添加文件夹并完成第一次备份。

## 前提条件

- 已完成 [安装](./installation)

## 流程总览

1. 创建配置
2. 添加要保护的文件夹
3. 执行备份（单个或全部）
4. 在历史页验证备份结果

## 1) 创建配置

1. 打开首页，点击 **「新建配置」**。
2. 选择配置类型：
   - **Default**：通用场景（推荐新手）
   - **Encrypted**：加密配置（后续还原需密码）
   - 其他插件类型：由已启用插件提供（如 Minecraft 相关类型）
3. 输入配置名称。
4. 确认后会生成配置卡片，点击卡片进入 **管理** 页面。

:::tip 关于插件类型
配置类型由插件动态扩展；如果你没启用插件，通常只会看到 Default 和 Encrypted。
:::

## 2) 添加文件夹

在 **管理** 页面点击 **「添加文件夹」**，有三种方式：

- **选择单个文件夹**：最常用。
- **批量导入子文件夹**：把所选目录的一级子目录全部加入当前配置。
- **扫描（插件自动发现）**：由插件识别特定目录结构并自动发现可管理文件夹。

添加完成后，文件夹会出现在列表中，状态初始通常为“从未备份”。

> **提示：** 每个文件夹是软件操作的最小单元。一个配置可包含多个文件夹，它们共享同一套备份规则。

## 3) 执行首次备份

你可以二选一：

- **一键备份所有**：执行当前配置下全部文件夹。
- **备份此文件夹**：先选中某个文件夹，再只备份该项（可填写“本次备份注释”）。

执行后可在“备份任务”页面看到进度，完成后会写入历史记录。

## 4) 验证备份结果

1. 在管理页选中文件夹，点击 **「历史版本」**。
2. 检查是否出现新的历史条目（含时间、类型、大小）。
3. 可点击“查看”按钮在资源管理器中定位备份文件。

## 可选：首次备份前的建议设置

在“配置设置”中可先做两项：

- **备份模式**：全量 / 智能增量 / 覆写备份。
- **无变更时跳过备份**：减少无意义备份文件。

## 常见问题

- 点了备份没看到历史？先确认目标路径已设置且可写。
- 插件扫描没发现内容？确认插件已启用，并选择插件支持的目录结构。

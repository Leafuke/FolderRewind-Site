# 首次还原

学习如何将文件夹恢复到任意历史状态，并理解两种还原模式的区别。

## 操作步骤

### 1. 进入历史页面

1. 打开 FolderRewind 主界面。
2. 点击目标配置卡片，进入配置详情。
3. 选择要还原的文件夹，查看其 **历史时间轴**。

### 2. 选择还原点

在时间轴中，你会看到所有历史备份按时间排列。每个备份条目显示：

- 备份时间
- 文件大小
- 压缩包名称

点击选择你想要还原到的时间点。

### 3. 执行还原

1. 选中目标备份后，点击 **「还原」** 按钮。
2. 在确认对话框中选择还原模式：
   - **Clean 还原（清空目标）**：先清理目标目录再还原，最推荐。
   - **Overwrite 还原（覆盖）**：直接覆盖同名文件，可能保留旧文件。
3. 确认操作，等待还原完成。

:::info 加密配置
如果配置类型为加密配置，执行还原前会要求输入密码验证。
:::

:::caution 注意
Clean 模式会删除目标目录中的现有内容（白名单规则除外）。
如果你不确定，建议先做一次手动备份，或在配置中启用“还原前自动备份”。
:::

### 4. 验证还原结果

检查文件夹内容，确认文件已恢复到预期状态。

## 建议先配置两项“保险设置”

在配置设置的“还原策略”中：

- 开启 **还原前自动备份**：误操作时可回退。
- 配置 **还原白名单**：在 Clean 模式下保留指定文件/文件夹。

## 失败排查

- 备份文件不存在：历史条目对应压缩包可能已被手动删除或移动。
- 密码校验失败：确认输入的是该配置的正确加密密码。
- 还原后与预期不符：检查是否误用了 Overwrite 模式，或白名单保留了旧文件。

# 首次还原

学习如何将文件夹恢复到任意历史状态。

## 操作步骤

### 1. 进入历史时间轴

1. 打开 FolderRewind 主界面。
2. 点击目标配置卡片，进入配置详情。
3. 选择要还原的文件夹，查看其 **历史时间轴**。

### 2. 选择还原点

在时间轴中，你会看到所有历史备份按时间排列。每个备份条目显示：

- 备份时间
- 文件大小
- 压缩包名称

点击选择你想要还原到的时间点。

### 3. 执行还原

1. 选中目标备份后，点击 **「还原」** 按钮。
2. 选择还原位置：
   - **原位还原** — 直接覆盖当前文件夹内容
   - **还原到指定位置** — 解压到自定义目录
3. 确认操作，等待还原完成。

:::caution 注意
原位还原会覆盖当前文件夹中的文件。建议在还原前先执行一次备份，以防万一。
:::

### 4. 验证还原结果

检查文件夹内容，确认文件已恢复到预期状态。
